# R analysis project theme

Creates a theme that scaffolds a simple R analysis project with
`main.R`, `README.md`, and `.gitignore`.

## Usage

``` r
r_analysis()
```

## Value

A `prefab_theme` object.

## Examples

``` r
r_analysis()
#> <theme> 3 steps
#> • file → main.R (skip)
#> • file → README.md (skip)
#> • text → .gitignore (union)
#> ℹ Apply with `use_theme()` or `create_project()`
```
