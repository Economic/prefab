# prefab: Opinionated Project Scaffolding via Composable Themes

Provides a composable theme system for project scaffolding. Themes are
ordered lists of steps that deploy files, inline text, or execute
functions, with per-file merge strategies for handling pre-existing
content. Includes pre-set themes for R analysis projects, targets
workflows, and Claude Code agent configuration.
